About CareerToolFinder

An independent comparison site helping job seekers pick the right tools — resume builders, interview prep, job boards, and LinkedIn growth. Built by one person. Fully transparent about how it works.

Who built this

CareerToolFinder is built and maintained by John Ogoina, an independent digital builder based in Lagos, Nigeria, operating under the GitHub organization aiopentec. It's one of a small portfolio of zero-cost, automated comparison sites, alongside Open Source Alternative Finder and Email Tool Compare.

How this site works

  1. Tool data lives in a structured file. Every tool's pricing, pros, cons, and positioning is stored in a single data file, not scattered across hand-written pages.
  2. A Python build script generates every page. Comparison pages, category pages, and alternatives hubs are all templated and generated automatically from that data.
  3. GitHub Actions rebuilds the site automatically on every update and on a weekly schedule, then deploys to GitHub Pages.
  4. No page is written by hand. Adding a tool or updating pricing means editing one data entry — the build script regenerates every affected page.

What this site does not do

  • No one has personally tested every tool listed here in a real job search. Pros, cons, and pricing are based on publicly available information at the time of writing.
  • Pricing changes over time. Always check the tool's own website before subscribing or purchasing.
  • ATSKILLER is our own product and is disclosed as such wherever it's recommended — see the affiliate disclosure below.

Affiliate & commercial disclosure

ATSKILLER, recommended on resume and ATS tool pages, is a product built by the same person who runs this site. We do not currently run third-party affiliate links, but may add them in the future — this page will be updated if that changes.
